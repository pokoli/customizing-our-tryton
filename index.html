Software requirements

Must

  1. Python (of course)
  2. pip
  3. sqlite
  4. python-lxml installed (or development headers to compile it)
  5. Mercurial

Recomended

  1. Use a virtualenv
  2. Do not use windows (if possible)
  3. Use python2.7 (otherwise you should run 2to3)

What is tryton?

  1. Application Platform
  2. Usually for building erps (but not limited)

But also:

  1. Community
  2. Foundation
  3. No lock-in

Tryton Architecture

3 Thiers:

  1. Application Server (trytond)
  2. 2 Clients: GTK (tryton) and Javascript(sao)
  3. DBMS: PostgreSQL/SQLite/MySQL/(Oracle)

Designed for modularity

Each module adds:

  1. Dependencies
  2. Models
  3. Views
  4. Database configuration
  5. Static Data

What is a tryton module?

A Python Package
